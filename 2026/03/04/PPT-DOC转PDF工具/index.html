一、工具介绍

写了一个小工具，可以把 PPT 和 Word 文档批量转换成 PDF。

功能：

  • 支持 .ppt、.pptx、.doc、.docx 转换为 PDF
  • 单文件转换 / 文件夹批量转换
  • 静默模式（后台运行，不弹窗）

二、环境配置

环境要求

  • Windows 系统
  • Microsoft Office (Word + PowerPoint)
  • Python 3.x
  • Python 包: pywin32、tqdm

常见问题

点击展开查看

Q: 转换失败怎么办？

A: 尝试加上 -v 参数查看错误信息，或确保 Office 软件正常运行。

Q: 速度慢吗？

A: PPT/DOC 转换需要启动 Office 进程，每个文件约需 5-15 秒。

Q: 可以批量转换吗？

A: 可以，直接运行 python convert_to_pdf.py -i 文件夹路径 即可批量转换该文件夹下所有 PPT/DOC/PDF 文件。
